## How It Works

1. **Choose a Topic**: Search for anything or select from pre-loaded topics
2. **Select Your Level**: Pick the depth of explanation you want
3. **Explore**: Get AI-generated content tailored to your learning level
4. **Learn**: Read explanations optimized for your knowledge level

## Technology Stack

- **Frontend**: Pure HTML, CSS, JavaScript (no frameworks needed)
- **AI**: Claude API (Anthropic) for content generation
- **Hosting**: Azure Static Web Apps
- **CI/CD**: GitHub Actions (automatic)

## Learning Levels

- **Basic**: Simple language, everyday analogies, foundational concepts
- **Intermediate**: More technical terms, assumes basic knowledge
- **Advanced**: Technical language, complex concepts, strong foundation required
- **Expert**: Highly technical, cutting-edge topics, expert background assumed

## Configuration

The app uses the Anthropic API for content generation. The API calls are made directly from the browser with no backend required.
